Code for Russia

Сообщество IT-специалистов, создающих проекты и технологии для становления свободного демократического общества в России

{% include socials.html %}

Code for Russia ставит целью разработку проектов и технологий с открытым исходным кодом по следующим основным направлениям.

Гражданские технологии

Проекты, усиливающие горизонтальные связи в обществе, а также технологии, упрощающие взаимодействие волонтеров, НКО, экспертов и др. граждан

Инструменты прямой демократии

Технологические проекты, направленные на снятие барьеров для обеспечения механизмов прямой демократии и прозрачного государства - проведения голосований, инициации законов, формирования экспертных групп, открытой финансовой отчетности, юридического делопроизводства на основе smart-контрактов и др.

Информационно-просветительские проекты

Проекты, информирующие граждан об их гражданских правах и возможностях. Проекты, формирующие представление о правовом обществе с функционирующими демократическими институтами, работающими бок о бок с современными технологиями.

Технологии для расследовательской журналистики

Технологии, помогающие независимым медиа проводить проверку фактов и расследовать преступления на основе обработки и анализа открытых источников данных.
